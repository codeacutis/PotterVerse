# PotterVerse

Projeto desenvolvido na disciplina de Programação Frontend da UNIVAS.

**Alunos:** Guilherme Augusto Alves de Faria & João Pedro Rodrigues Vilas Boas 
**API utilizada:** [HP API](https://hp-api.onrender.com/) — [Documentação](https://hp-api.onrender.com/)

---

## Instalação e execução local

**Pré-requisitos:** Node.js instalado.

```bash
# Clone o repositório
git clone https://github.com/guidefar/potterverse.git
cd potterverse

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em `http://localhost:5173`

---

## Funcionalidades

- **Autenticação** — tela de login simples com validação e rotas privadas
- **Personagens** — listagem com busca por nome e filtro por casa de Hogwarts, com página de detalhe de cada personagem
- **Feitiços** — listagem com busca por nome e página de detalhe de cada feitiço
- **Responsividade** — layout adaptado para mobile e desktop
